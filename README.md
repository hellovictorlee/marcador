# Bookmarks

minimal bookmark manager

## Install
	$ git clone git@github.com:joajfreitas/bookmarks.git
	$ cd bookmarks
	$ make install

You may need root privileges for the last command

## Usage
CLI interface
	$ bookmarks -h

Rofi interface
	$ rofi-bookmarks
