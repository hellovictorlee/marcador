# Bookmarks

minimal bookmark manager

## Install
	$ git clone https://github.com/joajfreitas/bookmarks.git
	$ cd bookmarks
	$ make install

You may need root privileges for the last command

## Usage
CLI interface

	$ bookmarks -h

Rofi interface

	$ rofi-bookmarks

## Thanks
